Fonctionnalites realisees dans cette version 7 :



        - 4.1 Attaque de monstre :
        
                Criteres de validation testes dans la classe TestsDeplacementAleatoireMonstre :
                        
                        + Lorsque le mosntre est à coté du joueur, il attaque sans se déplacer
                        + Si le monstre est loin du joueur il se déplace mais n'attaque pas 
                        
                        
-----------------------------------------------------------------------------------------------------------------------------------------------------
        
        - 4.2 Attaque du jouer :
        
                Criteres de validation testes dans la classe TestsDeplacementAleatoireMonstre :
                
                        + L'attaque est effectuée quand le joueur appuie sur la touche
                        + Tous les monstres a coté du joueur prennent des dégats sauf ceux qui sont éloignés
                        + S'il n'y a pas de monstre autour l'attaque a quand même lieu mais le joueur ne se déplace pas et les monstres ne prennent pas de dégats

-----------------------------------------------------------------------------------------------------------------------------------------------------
        
        - 4.5 Mort des monstres :
        
                Criteres de validation testes dans la classe TestsDeplacementAleatoireMonstre :
			
			+ Les monstres ont des points de vie qui ne peuvent pas être négatifs
			+ Quand le monstre n'a plus de points de vie il disparait 
			+ Le héros attaque avant le monstre
